出国留学之匹兹堡大学——申请篇

发表于 2017-12-18 | 分类于 Others |
| 字数统计 1,384

　　这个系列其实从我拿到学校offer之后就一直是打算着开学之后就写，但是万万没想到开学之后实在是太忙，现在轮到第一学期的寒假了，终于也是闲了下来，可以有时间补上。希望下学期没有这么忙了吧（不过话是这么说但是好像我还是给我下学期找了很多事，不过还好压力没有上学期这么大了，期待日子好过一点吧）

　　申请的时间因为已经相隔了一年以上，也只能尽我所能地回忆一些了，希望能给将来要申请的同学一点帮助吧。

　　先是自报家门一下三维背景，我的话从小学五年级开始搞计算机，初中一等奖保送的高中，高中一等奖保送的上交。不过从高一拿了一等奖之后就没怎么好好学习了，直接导致的后果就是我高二高三都是二等奖，而且成绩是真的不怎么样。本科读的是软件工程，基本就是混，算法数据结构自然是秒的，但是以前没接触过的计算机系统至今都很是头疼，而且因为高数和概统重修多次直接导致了丧失保研的可能，考研当然后果也就是数学挂了，所以转向了出国留学这条道路。

　　然后也因为基本高中开始就是个学渣了，最后的结果也不怎么样，在匹兹堡大学的information science专业，不过现在读着还算满意。

　　我因为是先参加的考研，然后才开始的准备留学，所以相当于当中gap了一年，正规的做法应该是大三开始申请，大四收offer，然后大四结束就可以直接去读研究生了。我翻看了一下邮件我应该是2016年8月份找的留学中介，威久留学。中介还是不错的，可以帮忙省下很多心，但是一般也不要指望中介能帮你写出多么华丽的CV推荐信之类的，就一般中等水平吧，如果期望值很高的同学还是建议自己弄。

　　我的时间线大概是：

  • 之前似乎有一次托福，不是记得很清楚，但是应该一共考了3次
  • 2016-05-07 第一次托福，分数86
  • 2016-07-17 第一次GRE，分数145+168+3
  • 2016-08 找的中介
  • 2016-09-24 第二次托福，分数不记得了，不如第一次
  • 2016-10-30 第二次GRE，分数96
  • 2016-10 ~ 2016-11 申请
  • 2017-04 PITT offer
  • 2017-06-22 签证check，上海梅陇镇广场
  • 2017-07-14 签证issue

　　总的来说时间线还算比较正常，然后来细说一点细节。

　　托福和GRE这玩意儿没啥好说的，坚持地下来背单词就赢了。（我就是没坚持下来，输了=。=）写作文的话个人推荐一个我新东方老师出的卡片册，叫托福写作一本通之论证论据素材100例，想考100左右的有它还是很管用的，要再往上去这个就帮不到你了。

　　申请季一般都集中在9月到12月，越早越好，分数越高越好，我GPA只有3.1所以……但是这么看来的话我三维这么低还是很轻松地录了PITT的IS专业可见这个专业确实不咋滴，但是垃圾学校垃圾专业自有它的好处这个在找工篇我会细讲，而且这个专业一半印度人一半中国人零星美国人也是侧面印证了这一点。

　　出国留学申请项目和工作经历到底重不重要这个问题嘛，要我说我觉得到匹大这个档次左右就不重要了，再往上可能会真的看，因为竞争比较激烈，不用这个分不出高低。不过国内大家都知道或多或少这种东西都有点造假成份在里面，我觉得就不要过度重视这个了。

　　关于签证的话，首先预约签证的网就是个垃圾，上海的更垃圾，高峰期注册页面加载不出来是常有的事，打领事馆电话更是坑，你根本听不到对面的自动语音在说什么，说一个字跳三个字听不清的。这个时候就要说一下中介的好了，因为她帮我直接在香港注册了一个账号然后打到领事馆帮我转成了上海的账号。（神一样的操作）check的话没什么道理，安然受之吧，我就是一上来啥都没说她就开始问我收材料了，材料齐全一般一个月左右可以签出，急也没用。

　　全部结束了之后记得去打学校规定的疫苗，我们学校规定的是MMR和水痘，上海的出入境检疫局比较厉害，可以测水痘抗体（其他大部分城市不可以），我直接测了全是阳性然后就啥都没打。另外疫苗和抗体是不用拿着签证就可以弄的（就比如我这种check的），体检似乎一定要签证，具体的可以直接跑过去问问，每个地方规定都不一样，当面问最稳妥。
